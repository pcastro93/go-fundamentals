# go-fundamentals

A structured repository documenting my Go learning process through focused exercises, patterns, and small implementations.

## Purpose

This repository is used to:

- Practice core Go concepts
- Explore idiomatic Go patterns
- Implement data structures and algorithms
- Experiment with concurrency
- Write and test production-style code

## Structure

The project is organized by topic:

- `strings/` – string manipulation exercises
- `loops/` – loops fundamentals
- `concurrency/` – goroutines, channels, synchronization
- etc.

## Goals

- Write idiomatic Go
- Prefer simplicity over abstraction
- Follow Go conventions and tooling
- Keep code readable and well-tested
